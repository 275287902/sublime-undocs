Exploring the API
*****************

A quick way to see the API in action:

#. Add ``Packages\Default`` (**Preferences | Browse Packages...**) to your project.
#. ``CTRL + SHIFT + F``
#. Enter ``*.py`` in the **In Files:** field
#. Check ``Use Buffer`` option
#. Search API name
#. ``F4``
#. Study relevant source code
